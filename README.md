# Alt'Ollama

The app for generating alt-text for image/s using LLMs with image processing support.

**The are two ways to generate alt-text for image/s:**

### Local (The models installed on the machine will be used)

1. Install https://ollama.com/ on your machine
2. Install and run model `llava-llama3` or `moondream` (model has to support image processing)
3. Install packages and run app
4. Use tab "Ollama Local"

### Cloud (The models from Ollama Cloud will be used)

1. Install packages and run app
2. Use tab "Ollama Cloud"

Please note: App will only works in DEV mode (e.g. npm run dev).
